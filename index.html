Autenticação

📋 Como pegar os tokens:

  1. Abra redacao.pr.gov.br
  2. Faça login normalmente com sua conta Google
  3. IMPORTANTE: Após o login, copie o link completo da URL (barra de endereço do navegador)
  4. O link deve começar com: https://redacao.pr.gov.br/#state=...
  5. Pressione F12 para abrir o DevTools
  6. Vá na aba Application (ou Aplicativo)
  7. No menu lateral, clique em Local Storage → https://redacao.pr.gov.br
  8. Procure a chave token e copie o valor completo
  9. O token começa com eyJ e tem 3 partes separadas por pontos
💡 Dica: O token expira após algumas horas. Se der erro de autenticação, pegue um novo token!
